QMSS_in_R: package, code and data

This repository contains the R package, code, and data for Greg Eirich's QMSS courses at Columbia University. Below you will find instructions for installing the QMSS package and downloading the data and R code for the courses.

Installing & loading the QMSS R package

Run the following three lines of code in R to download the "devtools" package, load the "devtools" package, and use the install_github function to install the QMSS package:

install.packages("devtools", dependencies = TRUE)

library(devtools)

install_github("jgabry/QMSS_in_R/Package/QMSS")

After installation has completed run library(QMSS) to load the package. Run help(package="QMSS") to open the R documentation for the package.
